Before Submitting Complete These Steps:
Follow this GIF:
You must put in the contract's description the following code:
{{ship_order.uid}}
Or the same instructions but in Text:
  1. First make sure you haven't already ordered a ship today and that if you are ordering a Maller/Thorax ProtoType/Vexor Navy Issue/Caracal Navy Issue you haven't already ordered one this week.
  2. Next if you are paying by isk for a non-free ship (T7 or T8) lookup the contract price by using the !quote shipname command in #boats-r-us
  3. Now start making a contract ingame, it's visibility must be set to the alliance.
  4. Next select Jita IV - Moon 4 ITC as the station location. If you don't see this option it is because you don't have an item in that station. Buy something cheap on the market to get an item there first.
  5. You must put in the contract's description the following code:
    {{ship_order.uid}}
  6. Make sure the expiry is set to 1 week, or 2 weeks for a battleship.
  7. Then scroll down, click add item and find the ship you want and add it.
  8. Finally you can create the contract.
  9. If you mess up any of these steps, your ship order might not be filled as the industry team will not be able to find or verify your contract.
